# Nu Wang — Page de vente

Page de vente pour le programme d'accompagnement Nu Wang (女王) par Line Zhu / Phoenix Shift.

## Stack

- HTML/CSS/JS pur (fichier unique)
- Hébergé sur GitHub Pages
- Domaine : `nu-wang.linezhu.com`
- Paiement : redirige vers `paiement.linezhu.com` (Systeme.io + Stripe)

Le site se déploie automatiquement via GitHub Pages depuis la branche `main`.
